# PoiBot V4

![Gradle CI](https://github.com/PoiCraft/PoiBot-V4/workflows/Gradle%20CI/badge.svg)

PoiBot V4 是一个基于 [mirai](https://github.com/mamoe/mirai) 的，提供基于 QQ群 的面向 `BedrockX for Minecraft Bedrock` 设计的服务器管理机器人
